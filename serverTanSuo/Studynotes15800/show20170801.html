第二节：java语言概述
SUN公司的 1995年
1999年 java被分为了J2SE J2EE J2ME
现在叫做JavaSE JavaEE JavaME
被Oracle收购了
目前 我们用Java 1.8（2014年）
1、桌面应用程序 被.net平台 C#挤占了市场
2、手机App JavaME
3、Web应用程序 JavaEE
Java开发服务器
大数据 BigData Hadoop Python-数据分析

Java主要的技术特点：
a、简单性：由c++衍生而来，但是比c++简单，抛弃了指针、头文件
b、面向对象(OO)：OOAD OOA OOD OOP
c、跨平台（一种与平台无关的语言）!!!!!
一次编译，处处运行。
如何实现？————Java虚拟机 JVM
JDK—Java开发工具（开发人员） —javac
JRE—运行编译后的class文件 —java
JVM—翻译
对于不同的操作系统有不同的翻译
在不同的操作系统上有不同的虚拟机
JVM是JRE的一部分
d、健壮性和安全性：垃圾回收机制
e、多线程：Java对多线程的支持非常强大，很短的代码就可以完成很强大的功能

背诵Java文件的执行过程：
Java文件---javac--->.class文件(字节码文件)
---java --->类装载器(装载类)---------\
------>字节码校验器--------------------JVM
------>解释器------------------------/
------>操作系统

第三节：java开发运行平台
jdk安装路径不要有中文、特殊符号、空格
JAVA_HOME jdk安装根目录
PATH %JAVA_H0ME%\bin
CLASSPATH JDK1.5以后不需要配置该环境变量

第四节：第一个java程序
执行java文件只能执行有main方法的java文件。
一个项目至少有一个main方法，其他所有类的方法都是直接或间接的通过main方法调用。
1、一个java文件中可以写一到多个类。
2、编译后的class文件个数和java文件中类的个数一致，而且class的文件名和类名相同。
3、一个java文件中最多有1个public修饰的类，而且java的文件名必须和public修饰的类名一致
4、建议一个java文件中只写一个类。

第五节：java语法基本元素
1、java标识符：
可以起名字的地方
命名规范：
由字母、数字、美元符号$、下划线组成
不能由数字开头
严格区分大小写，没有长度限制
不能和关键字重复
约定俗成：
见名知意 getUserByUserName
驼峰命名法：如果一个标识符有多个单词醉成，则从第二个单词开始，每个单词的首字母大写
类名首字母大写 class Student
包名全部小写
常量名全大写
关键字：
40多个，被系统赋予了特殊含义，由全小写组成
变量：存储数据的位置
java数据类型：（Java是强数据类型语言）
四类八种基本数据类型和三种引用数据类型
四类基本数据类型：
整型：byte short int long
浮点型：float(单精度 4字节) double(双精度 8字节) 默认double
字符型：char
布尔型：boolean
三种引用数据类型：
类
接口
数组
int num = 10;
byte 1字节 -128 ~ 127
short 2字节 -2^7 ~ 2^7-1
int 4字节 -2^15 ~ 2^15-1
long 8字节 -2^31 ~ 2^31-1

整型的默认数据类型是int，但是如果这个int类型的数值比较小的时候，会自动转换为byte或者short为其变量赋值
小类型的值向大类型赋值，会自动转化数据类型
大类型的值向小类型赋值，要报错，需要强制转换，会存在溢出
如果声明一个long类型的整数，在其后面加l或L，建议大写

字符型：
char值使用单引号包裹
char和整形可以相互转化
char可以自动转换成int类型直接参与运算
布尔型
boolean true false;
String是引用数据类型，引用数据类型变量内存储的是地址！

变量分为成员变量(方法外)和局部变量(方法内、块内、形参)
变量的作用范围：最近的一组大括号
变量必须先声明后赋值再使用
在变量的作用范围内不能重名

常量：常量的值一旦确定不能修改，由final修饰
final int CONSTANTEST = 10;

字面量 用于表达源代码中一个固定值的表示法

字符串常量、逻辑常量、字符常量... --->常量池

2、java运算符和表达式：
算数运算符（+ - * / % ++ --）

浮点数除以0，编译通过，运行结果Infinity(无穷大)
Java中没有四舍五入，10/3=3;11/3=3;12/3=4;
取模（取余）
