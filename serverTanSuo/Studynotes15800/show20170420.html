代码示例：
        /**
		 * 100元买100只鸡--公鸡9元2只--母鸡3元1只--小鸡1元5只--每种至少1只--求出有多少种组合
		 */
		// 总钱数 总只数
		double money = 100;
		int count = 100;
		// 公鸡 母鸡 小鸡的单价
		double moneyGJ = 9.0 / 2;
		double moneyMJ = 3.0;
		double moneyXJ = 1.0 / 5;
		// 组合数
		int sum = 0;
		// i表示公鸡的数量 j表示母鸡的数量 k表示小鸡的数量
		for (int i = 1; i < money / moneyGJ; i++) {
			for (int j = 1; j < money / moneyMJ; j++) {
				for (int k = 1; k < money / moneyXJ; k++) {
					double temp = i * moneyGJ + j * moneyMJ + k * moneyXJ;
					if (i + j + k == count && temp == money) {
						System.out.println(i + " " + j + " " + k);
						sum++;
						break;
					}
					if (i + j + k > count || temp > money)
						break;
				}
			}
		}
		System.out.println(sum);

2、数组的选择排序
冒泡排序算法
冒泡法是运用遍历数组进行比较，通过不断的比较将最小值或者最大值一个一个的遍历出来。
选择排序法是将数组的第一个数据作为最大或者最小的值，然后通过比较循环，输出有序的数组。
插入排序是选择一个数组中的数据，通过不断的插入比较最后进行排序。

                args[i]=args[j];
                args[j]=temp;

                }

                }

Produce类：
        /**
         * 定义一个商品类，商品在不同的季节有不同的价格
         *
         * @author guowh
         *
         */
        public class Produce {
            // 商品价格
            private double price;

            public double getPrice() {
                return price;
            }

            public void setPrice(double price) {
                this.price = price;
            }

            /**
             * 改变商品价格，如果是春季 商品价格加价50% 如果是夏季 商品价格减少50%
             *
             * @param ch表示季节
             *            c=春 x=夏 q=秋 d=冬
             */
            public void changePrice(char ch) {
                switch (ch) {
                case 'c':
                    price = price * 1.5;
                    break;
                case 'x':
                    price = price * 0.5;
                    break;
                default:
                    break;
                }
            }
        }
